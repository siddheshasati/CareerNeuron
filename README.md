## ⚙️ How It Works (Real-Time Pipeline)

CareerNeuron utilizes a "Neural Feedback Loop" to ensure every piece of advice is grounded in current market data:

1.  **Input Layer**: You upload your resume or paste a LinkedIn job description.
2.  **Neural Processing**: The **Kryvence Backend Engine** (powered by Gemini) analyzes the text for intent, skills, and tone.
3.  **Real-Time Benchmarking**: The system scrapes live industry trends to see how you stack up against other candidates.
4.  **Actionable Output**: Within seconds, you receive a dynamic dashboard with a clear "Gap Analysis" and optimization steps.

## 🛠️ Installation & Local Setup

If you wish to contribute or run the environment locally, follow these steps:

### 1. Clone the Repository
```bash
git clone [https://github.com/your-username/CareerNeuron.git](https://github.com/your-username/CareerNeuron.git)
cd CareerNeuron
2. Set Up Virtual Environment
Bash
python -m venv venv
# Windows:
.\venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
3. Install Requirements
Bash
pip install -r Requirements.txt
4. Configuration
Create a .streamlit/secrets.toml or .env file:

Code snippet
GEMINI_API_KEY = "YOUR_API_KEY_HERE"
5. Run the Application
Bash
streamlit run Main.py
